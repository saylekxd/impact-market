Stripe Payment API Server

API Endpoints

  • POST /api/create-payment-intent - Create a Stripe payment intent
  • POST /api/create-checkout-session - Create a Stripe checkout session
  • POST /api/payment-info - Update payment information in the database
  • GET /api/test - Test if the API is working
